# dousheng
字节第五届青训营大作业-极简版抖音
## 基本介绍：

### 1.1 项目架构

本项目采用 MVC 分层设计模型分离模型层、视图层和控制层，从而降低代码的耦合度，提高项目的可维护性。

使用 Gin 作为 Web 框架，Redis 作为缓存框架，MySQL 作为持久层框架。

```
         ┌─────────┐      ┌─────────┐      ┌─────────┐
  ──req──►         ├──────►         ├──────►         │
         │   Gin   │      │  Redis  │      │  MySQL  │
  ◄─resp─┤         ◄──────┤         ◄──────┤         │
         └─────────┘      └─────────┘      └─────────┘
```

### 1.2 功能介绍

- **视频：视频推送、视频投稿、发布列表**
- **用户：用户注册、用户登录、用户信息**
- 点赞：点赞操作、点赞列表
- 评论：评论操作、评论列表
- 关注：关注操作、关注列表、粉丝列表

### 1.3 技术选型

- Gin
- Gorm
- MySQL
- Redis
- ffmpeg（获取视频封面）
